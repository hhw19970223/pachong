# Skills 广场 - AI技能发现平台

## 🌟 功能特点

- **📊 实时数据展示** - 从 Skills.sh 实时获取最新技能数据
- **🔍 智能搜索** - 支持技能名称、描述、作者搜索
- **🏷️ 多维过滤** - 按下载量、星标、安全状态过滤
- **📱 响应式设计** - 完美适配桌面端和移动端
- **⚡ 快速安装** - 一键复制安装命令
- **🎨 现代UI** - 美观的渐变设计和动画效果

## 🚀 访问地址

- **本地开发:** http://localhost:3001/
- **公网访问:** http://98.88.137.186:3001/

## 📋 数据源

从以下接口获取技能数据：
- **API 缓存接口:** `/api/skills/cache`
- **实时爬取接口:** `/api/skills/scrape?source=skills.sh`

## 💡 功能介绍

### 主要功能
1. **技能展示** - 以卡片形式展示所有AI技能包
2. **搜索功能** - 实时搜索技能名称、描述、作者
3. **分类过滤** - 按不同维度筛选技能
4. **统计信息** - 显示总技能数、总下载量、平均星标
5. **分页浏览** - 支持大量数据的分页展示

### 技能卡片信息
- **排名与基本信息** - 显示技能排名、名称、作者
- **描述** - 技能的详细描述
- **统计数据** - 周下载量、星标数、安全审计状态
- **操作按钮** - 一键复制安装命令、查看源码仓库
- **更新时间** - 最后更新时间

### 过滤选项
- **全部** - 显示所有技能
- **高下载量** - 周下载量 > 10,000 的技能
- **高星标** - 星标数 > 10 的技能  
- **安全认证** - 通过绿色安全审计的技能

## 🎨 界面特色

- **渐变背景** - 蓝紫色渐变背景，视觉效果出众
- **卡片悬停效果** - 鼠标悬停时卡片上浮，增强交互体验
- **实时统计** - 顶部显示技能总数、下载量、星标数统计
- **响应式布局** - 自适应不同屏幕尺寸
- **图标美化** - 使用 Font Awesome 图标库
- **毛玻璃效果** - 统计栏和按钮使用半透明毛玻璃效果

## 🔧 技术实现

- **纯前端实现** - HTML + CSS + JavaScript
- **现代CSS** - CSS Grid 布局、Flexbox、渐变、动画
- **异步数据获取** - Fetch API 获取后端数据
- **客户端分页** - JavaScript 实现分页功能
- **响应式设计** - CSS 媒体查询适配移动端

## 📝 使用说明

1. **浏览技能** - 页面自动加载并显示所有技能
2. **搜索技能** - 在搜索框输入关键词，支持实时搜索
3. **筛选技能** - 点击过滤按钮筛选特定类型技能
4. **安装技能** - 点击"安装"按钮复制安装命令到剪贴板
5. **查看源码** - 点击 GitHub 图标访问源码仓库
6. **分页浏览** - 使用底部分页按钮浏览更多技能

## 🌐 兼容性

- **移动端浏览器** - 支持iOS Safari, Android Chrome
- **功能降级** - 老旧浏览器的复制功能自动降级
